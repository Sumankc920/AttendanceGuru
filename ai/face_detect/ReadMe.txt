Welcome to the MTCNN Face Detection Model! This model allows you to detect faces in images using the MTCNN (Multi-Task Cascaded Convolutional Networks) algorithm. This Readme file will guide you through the process of running the model in a simple and straightforward manner.

Prerequisites
Before running the MTCNN Face Detection Model, ensure that you have the following prerequisites installed on your system:

Python (version 3.6 or later)
TensorFlow (version 2.0 or later)
OpenCV (version 3.4 or later)

Installation
	Clone the repository or download the source code files to your local machine.

	Install the required Python dependencies by running the following command:
		pip install -r requirements.txt
		
	Then run the following command :
		python main.py
		
Conclusion
Congratulations! You have successfully set up and run the MTCNN Face Detection Model. Enjoy detecting faces in your images and feel free to explore further improvements or customizations based on your needs.

If you have any questions or encounter any issues, please don't hesitate to contact us.

Happy face detection!
